# Love2D Proto

This directory is for desktop-only visual prototyping.

Purpose:
- Explore 3D-ish number styling quickly on desktop
- Validate material, depth, highlight, shadow, and motion language
- Avoid wasting device iterations for early visual exploration

Run:

```bash
love tools/love2d_proto
```

Controls:

```bash
Space  -> switch material preset
```

This is not a deployable device runtime. It is only a desktop prototyping layer.
